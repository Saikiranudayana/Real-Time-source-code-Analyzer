How It Works

1

Clone & Parse

GitPython clones the repo. LangChain's GenericLoader and LanguageParser extract Python source files into structured documents.

2

Embed & Index

RecursiveCharacterTextSplitter chunks the code. HuggingFace embeddings convert each chunk to vectors stored in ChromaDB.

3

RAG + Groq

Your question retrieves the most relevant code chunks. Groq's LLM synthesises a precise, context-aware answer in real time.
